=head1 NAME

C<check_nethz> - a Nagios plugin to check the health of nethz DB

=head1 DESCRIPTION

check_nethz is a Nagios plugin to check the health of nethz DB

=head1 VERSION

Version 1.1.0

=head1 SYNOPSIS

 check_nethz [OPTIONS]

 Options
  --user,u            DB user
  --password,p        DB password
  --dbname,d          DB name
  --oracle-home       ORACLE_HOME environment variable
  --help,-h,-?        this help message
  --verbose,-v        verbose output
  --version,-V        print version

=head1 REQUIRED ARGUMENTS


=head1 OPTIONS

  --user,u            DB user

  --password,p        DB password

  --dbname,d          DB name

  --help,-h,-?        this help message

  --version,-V        print version

  --verbose,-v        verbose output

=head1 EXAMPLE

 export TNS_ADMIN=/usr/lib/oracle/11.1/client/network/admin
 check_nethz -u mutator -p ******* -d dbinfra.bd.ethz.ch

check_nethz checks if nethz is OK

=head1 DIAGNOSTICS

You can specify multiple --verbose options to increase the program
verbosity.

=head1 EXIT STATUS

0 if OK, 1 in case of a warning, 2 in case of a critical status and 3
in case of an unkown problem
